# redes-cisco
🚀 Redes - Cisco (CCNA)

En este repositorio comparto mis apuntes, esquemas lógicos y prácticas de laboratorio desarrolladas a lo largo de mi formación en redes con tecnologías Cisco, enfocándome en comprender cómo viajan los datos y cómo estructurar topologías seguras y eficientes.

El objetivo de este proyecto es dominar los fundamentos del modelo TCP/IP y OSI, aprender a configurar dispositivos de red (routers y switches) y asimilar las bases de conectividad indispensables para la auditoría y la ciberseguridad.

🛠️ Tecnologías y Conceptos Aprendidos

    Arquitectura de Red y Capas: Comprensión de los modelos OSI y TCP/IP, analizando cómo se encapsulan los datos desde la aplicación hasta el medio físico y el comportamiento de los protocolos esenciales (ICMP, DHCP, DNS).

    Direccionamiento e Infraestructura: Configuración de redes IPv4, segmentación mediante máscaras de subred (Subnetting) y direccionamiento estático/dinámico para optimizar el tráfico de datos.

    Simulación de Topologías: Uso de herramientas de simulación para diseñar, interconectar y diagnosticar redes complejas antes de su implementación en entornos reales.
